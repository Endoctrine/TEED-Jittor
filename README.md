# Tiny and Efficient Edge Detector (TEED) - Jittor Implementation

这是对 [Tiny and Efficient Model for the Edge Detection Generalization](https://github.com/xavysp/TEED) 的复现。使用了 Jittor 深度学习库代替原论文中的 Pytorch 库。
